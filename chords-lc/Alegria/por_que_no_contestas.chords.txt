 (Acorde)    (Cuerdas)
    |       E A D G B E    Los números corresponden al traste de la guitarra
    |       | | | | | |    donde se pulsa la cuerda señalada, la letra'X'
    |       | | | | | |    significa que no se pulsa la cuerda indicada.
    |       | | | | | |
    |       | | | | | |
    Cm   :  X 3 5 5 4 3  ( Do menor )
    D    :  X X 0 2 3 2  ( Re Mayor )
    F    :  1 3 3 2 1 1  ( Fa Mayor )
    G#m  :  4 6 6 4 4 4  ( Sol menor Sostenido )
    A#   :  X 1 3 3 3 1  ( La Mayor Sostenido )
